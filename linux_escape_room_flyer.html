// Storyline

SYSTEM STATUS: COMPROMISED · USER: YOU

Im Jahr 2084 ist dein Linux-System das letzte freie Interface in einer von Konzernen kontrollierten Netzwelt. Eine KI hat dein System verriegelt – nur wer die richtigen Befehle kennt, kommt wieder raus.

Du hast nur ein Terminal, kryptische Logfiles und versteckte Hinweise im Dateisystem. Finde das Passwort, übernimm einen neuen Account, knacke eine verschlüsselte Datei und leg am Ende die Firewall lahm – oder bleib im digitalen Käfig zurück.

DEINE MISSION

1) Passwort finden · 2) User übernehmen · 3) Datei entschlüsseln · 4) Firewall „hacken“ und entkommen.

Ideal, wenn du Linux bisher nur aus der Theorie kennst oder für Prüfungen übst. Hier kannst du ohne Risiko ausprobieren, was passiert – und am Ende wirklich sagen: „Ich hab ein System gehackt“
